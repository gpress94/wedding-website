
# Image Setup

To ensure your photos appear on the website, please place them directly in this folder (`public/images/`). 

**Do not create subfolders.**

## Required Files

Please ensure your files are named exactly as follows (case-sensitive):

1. **`hero.jpg`** - The large main photo on the homepage.
2. **`venue.jpg`** - The photo for the Events page.
3. **`hotel.jpg`** - The photo for the Travel/Accommodations page.

## Optional Header Art
4. **`header-left.png`** - Transparent pencil art to the left of "Nicolle".
5. **`header-right.png`** - Transparent pencil art to the right of "Greg".

## Fonts
If using custom fonts, create a `fonts` folder in public (`public/fonts/`) and add:
- `Nazeefa.otf` or `Nazeefa.ttf`

## Troubleshooting

- Ensure the file extension matches exactly (e.g. `.jpg` vs `.jpeg`).
- Ensure the filenames are lowercase.
